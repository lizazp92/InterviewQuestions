What is a computer?

A computer is an electronic device that stores, retrieves, processes data and gives an output according to instructions provided by computer programs.

What is a program?

A program is a set of instructions that a computer follows in order to perform a particular task.

What is a server?

A server is a computer or system that provides resources, data, services, or programs to other computers, known as clients, over a network.

What is the internet?

The internet is a global network of interconnected computer networks that exchange information using the TCP protocol.

What is The World Wide Web?

The World Wide Web is all the public websites or pages that users can access on their devices through the internet. The websites are connected by links between them (hyperlinks) forming a web.

What is a web browser?

A web browser is an application for accessing and viewing content on the World Wide Web.

What is a web page?

A webpage is a single document that uses a unique URL and can be displayed in a web browser.

What is a cookie?

A cookie is information stored on the user's computer for session management, personalization and tracking purposes.

What is IDE?

IDE (Integrated Development Environment) is a an application that provides tools to write and test software.

What is DevTools?

DevTools is a set of web developer tools built directly into the browser that can help you edit pages on-the-fly and diagnose problems quickly.

Whats is a markup language?

A markup language is a computer language that definies the structure and presentation of a document.

What is HTML?

HTML is Hyper Text Markup Language for creating Web pages. HTML describes the structure of a Web page.

What is semantic HTML?

Semantic HTML or semantic markup is HTML that introduces meaning to the web page rather than just presentation.

What is the difference between the article and section elements?

The article element specifies independent, self-contained content. The section element defines section in a document.
